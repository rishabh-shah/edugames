# Code of Conduct

EduGames is a collaborative project for software used by children and families. We expect contributors and maintainers to act with respect, care, and professionalism.

## Expected Behavior

- Be respectful in code review, issues, chat, and docs.
- Give actionable feedback tied to the work, not the person.
- Assume good intent while still being honest about risks and mistakes.
- Protect privacy and avoid sharing sensitive personal or security details in public.
- Accept maintainer decisions about project scope, safety, and moderation.

## Unacceptable Behavior

- Harassment, bullying, threats, or discrimination
- Sexualized language or imagery
- Doxxing or sharing private information without consent
- Deliberate sabotage, intimidation, or bad-faith review behavior
- Publicly posting exploit details or sensitive child-safety concerns before maintainers can respond

## Scope

This code of conduct applies to repository activity, pull requests, issues, discussions, and project-linked communication spaces.

## Reporting

Report conduct concerns privately to the project maintainers. Do not open a public issue for harassment, personal safety, or other sensitive reports. If the concern is primarily a security issue, follow `SECURITY.md`.

## Enforcement

Maintainers may remove content, issue warnings, limit participation, or ban contributors who violate this code. Responses should be proportional to the behavior and prioritize community safety.
